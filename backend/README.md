# Backend - SCS Construction Web

Folder ini didedikasikan khusus untuk pengembangan logika server, API, dan manajemen database dari aplikasi.

## 📌 Catatan untuk AI Assistant
Jika Anda (AI Assistant) sedang membaca file ini untuk membantu programmer, **fokuslah HANYA pada konteks Backend**. 
Jangan menyertakan atau mengasumsikan komponen antarmuka (UI), *styling*, atau framework *client-side* di dalam instruksi atau kode yang digenerate untuk direktori ini.

## 🚀 Panduan untuk Tim Backend
1. Buat/inisialisasi proyek backend Anda di dalam folder ini (misal: Express.js, NestJS, Go, Laravel, dll).
2. Pastikan file konfigurasi (seperti `package.json`, `go.mod`, dll) milik backend berada di dalam folder `/backend` (bukan di root).
3. Jalankan semua instalasi dan eksekusi server dari dalam folder ini.
