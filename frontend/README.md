# Frontend - SCS Construction Web

Folder ini didedikasikan khusus untuk pengembangan antarmuka (UI/UX) dan sisi *client-side* dari aplikasi. 

## 📌 Catatan untuk AI Assistant
Jika Anda (AI Assistant) sedang membaca file ini untuk membantu programmer, **fokuslah HANYA pada konteks Frontend**. 
Jangan menyertakan atau mengasumsikan logika server, koneksi database, atau arsitektur backend di dalam instruksi atau kode yang digenerate untuk direktori ini.

## 🚀 Panduan untuk Tim Frontend
1. Buat/inisialisasi proyek frontend Anda di dalam folder ini (misal: Next.js, React, Vue, dll).
2. Pastikan file `package.json` milik frontend berada di dalam folder `/frontend` (bukan di root).
3. Jalankan semua instalasi dan command npm dari dalam folder ini.
